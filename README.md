## ✨ Key Features

### 🌗 Theme System
- Light Mode: Sky, clouds, sun, floating particles
- Dark Mode: Space, stars, nebula, shooting stars
- Smooth transition between themes

### 🖱️ Interactive UI
- Custom animated cursor
- Parallax mouse movement (dark mode)
- Glassmorphism UI effects
- Smooth hover animations

### 💼 Projects Showcase
- 3 featured projects with live demos
- Tech tags & descriptions
- GitHub project integration

### 📬 Contact System
- EmailJS integration
- Form validation
- Success popup animation

### ⌨️ Keyboard Shortcuts
| Key | Action |
|-----|--------|
| H | Home |
| A | About |
| S | Skills |
| P | Projects |
| C | Contact |
| Esc | Close Menu |

## 🔧 Setup

### Clone Repository

git clone  https://github.com/praveenkumar19/PORTFOLIO.git
Open Project
cd PORTFOLIO

Then open index.html in browser or use VS Code Live Server.

⚙️ Customization
Replace profile image → assets/passport-.png
Update projects → index.html
Change colors → CSS variables
Update logo → signature images
EmailJS keys → JavaScript config
