📌 Overview
I'm Praveen Kumar, a passionate Cyber Security student and Frontend Developer based in Chennai. This portfolio represents my digital presence and showcases my skills, projects, and journey in the world of web development and cybersecurity.

The website features an immersive user experience with:

🌗 Dynamic Theme Toggle – Light mode with animated sky & clouds, Dark mode with space & stars

🎨 Custom Interactive Cursor – Animated cursor with hover effects

📱 Fully Responsive – Optimized for all devices

⚡ Smooth Animations – Scroll-triggered reveals & transitions

🔧 Customization Guide
Change Profile Image
Replace assets/passport-.png with your own image.

Update Projects
Edit the project cards in the <section id="work"> section of index.html.

Modify Colors
Update CSS variables in :root and body.dark-mode sections.

Change Logo
Replace signature-dark-.png and signature-light-.png in assets/ folder.

Update EmailJS Keys
Modify the following in the JavaScript section:

javascript
emailjs.init("-z9wMh99rBs8RcmnS");  // Your Public Key
const serviceID = "service_pfgh0jt"; // Your Service ID
const templateID = "template_2s3npoa"; // Your Template ID
